SERVICES I OFFER

Design

During the design phase I run through wireframing and use design packages such as Photoshop and Illustrator. This is the initial step to projects such as logo design and web design. At this stage involvement and feedback from the client is necessary.

Development

At this stage of the process, combine the artwork that has been created and make it sing and dance into a webpage. Using technologies such as HTML5, CSS3, jquery and many other technologies that fit the scope of the project.

Browser Testing

Full scale browser testing is underdone to make sure the website does perform the way it is meant to. This stage is the assures that web page or app will be compatible to older browsers such as ie8 as well as a wide range of mobile and desktop browsers.

Responsive Design

This is an important step of the process of creating a viable website into today internet standards. The combination of using reliable frameworks such as Bootstrap 3 and Foundation 5 and some custom tweaking, to make sure websites can scale from mobile devices to high res desktops.
